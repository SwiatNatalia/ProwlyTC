ID.A1 Searching "Prowly" in Google graphics.

Preconditions
Opened Google.com web page

Test steps:
1. Search "prowly" in google browser
2. Click on All filters button
3. Choose Graphics filter

Expected result:
Page with pictures associated with Prowly brand is loaded.

ID.A2 Searching "Prowly" in Google Maps

Preconditions
Opened Google.com web page

Test steps:
1. Search "prowly" in google browser
2. Click on All filters button
3. Choose Map filter

Expected result:

ID.B1 Searching "Prowly" results in Google by Last hour

Preconditions
Opened Google.com web page

Test steps:
1. Search "prowly" in google browser
2. Click on Tools button
3. Change time filter from "whenever" to "Last hour"

Expected result: If results exist, page with results from last hour associated with prowly brand will be displayed. If there are no results an information that phrase was not found is displayed.
